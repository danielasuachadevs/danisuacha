# Daniela Suacha - Biodeprogramación Terapéutica

Landing page profesional para Daniela Suacha, enfocada en Biodeprogramación Terapéutica y sanación emocional profunda.

## 🚀 Características

- **Diseño Premium**: Interfaz moderna, limpia y responsive utilizando Tailwind CSS.
- **Secciones Modulares**:
  - **Hero**: Introducción impactante con CTA directo.
  - **Pain Points**: Identificación de desafíos comunes de los pacientes.
  - **Metodología**: Explicación de la Biodeprogramación.
  - **Proceso**: Pasos claros para la transformación personal.
  - **Pricing**: Opciones de inversión en bienestar.
  - **Testimonios**: Historias de éxito y transformación.
  - **Calendly Integration**: Sistema de agendamiento de sesiones integrado.
  - **FAQ**: Respuestas a dudas frecuentes.
- **Navegación Fluida**: Implementación de scroll suave para una mejor experiencia de usuario.
- **Optimización SEO**: Estructura semántica avanzada para buscadores.

## 🛠️ Stack Tecnológico

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Lenguaje**: [TypeScript](https://www.typescriptlang.org/)
- **Componentes**: React 19
- **Agendamiento**: [React Calendly](https://www.npmjs.com/package/react-calendly)

## 💻 Desarrollo

Primero, instala las dependencias:

```bash
npm install
```

Luego, corre el servidor de desarrollo:

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.

## 📦 Despliegue

Este proyecto está optimizado para ser desplegado en [Vercel](https://vercel.com/new).
